# CMEE Project: Neutral Model Assessment of IUCN Red List Criteria

## Overview

This project was completed as part of the MRes in Computational Methods in Ecology and Evolution at Imperial College London.

The aim of the project was to develop a mechanistic neutral model to investigate the performance of IUCN Red List assessment criteria under controlled conditions. By simulating species population dynamics without ecological differences between species, the model provides a baseline for understanding how extinction risk classifications emerge and whether observed patterns can be explained by stochastic processes alone.

The project combines ecological theory, simulation modelling, statistical analysis, and data visualisation to explore conservation assessment frameworks using reproducible computational methods.

## Research Objectives

* Develop a neutral simulation model representing species population dynamics.
* Generate synthetic datasets under controlled ecological assumptions.
* Evaluate the behaviour of IUCN Red List assessment criteria using simulated populations.
* Compare expected classification outcomes under neutral conditions.
* Investigate patterns in extinction risk assessments through computational experimentation.

## Methods

### Simulation Modelling

A mechanistic simulation framework was developed in Python to model population dynamics across multiple species over time. The model incorporates stochastic processes to generate realistic variation while maintaining neutral assumptions.

### Data Processing

Simulation outputs were processed and analysed using Python and R. Data cleaning, transformation, and aggregation workflows were designed to ensure reproducibility and scalability across multiple simulation runs.

### Statistical Analysis

Statistical methods were applied to:

* Explore population trajectories.
* Quantify extinction risk patterns.
* Compare classification outcomes.
* Assess variation across simulation scenarios.
